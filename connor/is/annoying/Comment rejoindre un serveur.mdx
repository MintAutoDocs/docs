## Aperçu

Ce guide explique comment rejoindre un serveur de jeu Minecraft en utilisant le panneau de contrôle reskiné de Ptyerodactyl.

## Étapes

1. Connectez-vous au panneau de contrôle en utilisant vos identifiants.
2. Une fois connecté, recherchez l'onglet "Serveurs" dans le menu principal.
3. Cliquez sur "Serveurs" pour afficher la liste des serveurs disponibles.
4. Recherchez le serveur Minecraft auquel vous souhaitez vous connecter et cliquez dessus pour accéder à sa page de configuration.
5. Sur la page de configuration du serveur, recherchez l'adresse IP et le port du serveur. Notez ces informations, car vous en aurez besoin pour vous connecter.
6. Ouvrez le jeu Minecraft sur votre ordinateur.
7. Dans l'écran d'accueil de Minecraft, cliquez sur "Multijoueur" pour accéder à la liste des serveurs.
8. Cliquez sur "Ajouter un serveur" pour ouvrir la fenêtre de configuration du serveur.
9. Dans la fenêtre de configuration du serveur, entrez un nom pour le serveur dans le champ "Nom du serveur".
10. Dans le champ "Adresse du serveur", saisissez l'adresse IP du serveur que vous avez notée précédemment.
11. Dans le champ "Port du serveur", saisissez le port du serveur que vous avez noté précédemment.
12. Cliquez sur "Terminé" pour enregistrer les paramètres du serveur.
13. De retour à l'écran de sélection des serveurs, vous devriez maintenant voir le serveur que vous avez ajouté dans la liste.
14. Sélectionnez le serveur et cliquez sur "Rejoindre le serveur" pour vous connecter.

Suivez ces étapes pour rejoindre un serveur Minecraft en utilisant le panneau de contrôle reskiné de Ptyerodactyl.
